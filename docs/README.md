# docs

Assets referenced by the main README.

## demo.gif

The README embeds `docs/demo.gif`. To (re)generate it:

1. Run the demo in a terminal:
   ```powershell
   ./demo.ps1            # installed clawvet
   ./demo.ps1 -Local     # repo build
   ```
2. Record the window with [ScreenToGif](https://www.screentogif.com) or the
   Windows Game Bar (`Win+G`).
3. Save the recording here as `docs/demo.gif`.

Keep it short (~20–30s) and use a readable terminal font/size.
